Statistics

This section displays various statistics and analytics created by the outbreak detection system for {{ disease_name }} disease. This involves displaying data retrieved by our outbreak detection system in the form of graphs and maps. The system retrieves a fixed number of latest tweets related to {{ disease_name }} disease and then filters all those tweets which contain legit information about an outbreak of {{ disease_name }} in some part of the world. This is done using a deep learning model which has been trained to classify whether a given tweet is relevant or not. Finally locations are extracted from the filtered tweeets and the statistics are prepared using these extracted locations.

The first plot gives an idea about the magnitude of {{ disease_name }} outbreak in various countries all around the globe. Higher the intensity of the colour of a country, Higher is the chance that the country is facing a {{ disease_name }} outbreak. This gives us a idea about which countries are currently affected by {{ disease_name }} and what is the magnitude of risk of a pandemic that each affected country is facing. The displayed world map is completely interactive and shows the number of relevant tweets corresponding to each affected country when the mouse pointer hovers over an affected country.



The second plot shows the top 10 countries which are at a risk of facing a {{ disease_name }} pandemic. This plot gives an idea about the current {{ disease_name }} hotspots all over the world. This can also be helpful to compare the risk of {{ disease_name }} pandemic in the countries with maximum number of dengue cases. Again the plot is completely interactive and the number of relevant tweets for the selected affected country will be displayed whenever the mouse pointer hovers over a bar corresponding to an affected country.
